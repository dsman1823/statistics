"Теория вероятностей и математическая статистика"
для специальности:

310304 «Информатика»		  
Оглавление | Программа | Теория | Практика | Контроль знаний | Об авторах

  1. Производятся три выстрела по мишени. Вероятность поражения мишени первым выстрелом равна 0,4, вторым - 0,5, третьим - 0,6. СВ \(X\) - число поражений мишени.
    (Ответ: \(M(X)=1,5, D(X) =0,73.\))
  2. Вероятность сдачи данного экзамена для каждого из четырех студентов равна 0,8. СВ \(X\) - число студентов, сдавших экзамен.
    (Ответ: \(M(X)=3,2, D(X)=0,64\))
  3. Из партии в 20 изделий, среди которых имеется четыре нестандартных, для проверки качества выбраны случайным образом 3 изделия. СВ \(X\) - число нестандартных изделий среди проверяемых.
    (Ответ: \(M(X)=0,6, D(X)=0,48\))
  4. Вероятность отказа прибора за время испытания на надежность равна 0,2. СВ \(X\) - число приборов, отказавших в работе, среди пяти испытываемых.
    (Ответ: \(M(X)=1, D(X) = 0,9\))
  5. Вероятность выигрыша по одному билету лотереи равна 1/6. СВ \(X\) - число выигрышных билетов из четырех.
    (Ответ: \(M(X) = 2/3, D(X) = 5/9\))
  6. Проводятся три независимых измерения исследуемого образца. Вероятность допустить ошибку в каждом измерении равна 0,01. СВ \(X\) - число ошибок, допущенных в измерениях.
    (Ответ: \(M(X)= 0,03, D(X) = 0,0297\))
  7. Непрерывная случайная величина задана функцией распределения. Найти математическое ожидание случайной величины, дисперсию и вероятность попадания СВ \(X\) в интервал \(0 \leqslant X \leqslant 3\)
          \( F(X) = \begin{cases} 0, x < 0 \\ \frac{1}{20}(x^2+x), 0 \leqslant x \leqslant 4, a=0, b=3 \\ 1, x>4 \end{cases} \)
    (Ответ: \(M(X)=2,53, D(X)=1,049, P(0 \leqslant X \leqslant 3 )=0,6\))
  8. Непрерывная случайная величина задана функцией распределения:
          \( F(x) = \begin{cases} 0, x\leqslant 0 \\ (x^3+x)/10, 0 \leqslant x < 2 \\ 1, x \leqslant 2 \end{cases} \)
    Найти \( M[X], D[X], \delta_x \), квантиль 0,75, моду и медиану.
    (Ответ: \( M[X]=1,4; D[X]=0,22; \delta_x=0,469. \))
  9. Плотность распределения вероятностей величины Y имеет вид:
          \( f(y) = \begin{cases} k y, 0 < y\leqslant 6 \\ 0, y \leqslant 0, y > 6 \end{cases} \)
    Определить: а) постоянную \(k\) ; б) математическое ожидание СВ \(Y\) ; в) математическое ожидание квадрата СВ \(Y\).
    (Ответ: а) 1/18; б) 4; в) 18.)
