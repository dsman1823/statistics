  1. Случайная величина \(X\) принимает значения \(X=n, (n=1,2,…)\) с вероятностью \(P \{X = n \} = 2^{-n}\). Найти функцию \(F(x)\) и вычислить \(P\{3\leqslant x \leqslant 6\}\).
    (Ответ: \(F(x)=\sum\limits_{i < x} 2^{-i}; P\{3\leqslant x \leqslant 6\} = 0.2344\))
  2. По двоичному каналу связи с помехами передаются две цифры 1 и 0. Априорные вероятности передачи этих цифр равны \(p(1)=p(0)=0,5\). Из-за наличии помех возможны искажения. Вероятности перехода единицы в единицу и нуля в нуль соответственно равны \(p(1/1)=p, p(0/0)=q\). Определить закон распределения вероятностей случайной величины \(X\) – однозначного числа, которое будет получено на приемной стороне:
    (Ответ: \( P\{X=0\}=(1-p+q)/2\), \(P\{X=1\}=(1-q+p)/2 \))
  3. Случайная величина \(X\) имеет следующую плотность вероятностей:
          \( f(x) = \begin{cases} x-\frac{a}{4}x^3, 0\leqslant x\leqslant 2 \\ 0, x\not\in[0,2] \end{cases} \)
    Определить \(a\), медиану величины \(X\) и функцию распределения \(F(x)\).
  4. Случайная величина X имеет следующую плотность вероятностей:
          \( f(x) = \begin{cases} 0.5x-a, 2\leqslant x\leqslant 4 \\ 0, x\not\in[2,4] \end{cases} \)
    Вычислить \(a\) и вероятности \(p(X < 0,2)\), \(p(X < 3)\), \(p(X \geqslant 3)\), \(p(X \geqslant 5)\).
  5. Случайная величина \(X\) имеет следующую функцию распределения:
          \( F(x) = \begin{cases} 0, x\leqslant 0 \\ a\sin 2x, 0 < x \leqslant \pi/4 \\ 1, x>\pi/4 \end{cases} \)
    Определить \(a\), плотность вероятностей \(f(x)\) и значение \(f(\pi/4)\).
  6. Дан график плотности вероятностей случайной величины \(X\):
    Записать \(f(x)\) в аналитической форме, определить функцию распределения \(F(x)\) и \(p(X < -0.5)\).
  7. Случайная величина X имеет следующую функцию распределения:
          \( F(x) = \begin{cases} 0, x\leqslant 0 \\ a x^3, 0 < x \leqslant 2 \\ 1, x>2 \end{cases} \)
    Определить \(a\), плотность вероятностей \(f(x)\), вычислить вероятность \(p(0 < x < 1.5)\).
